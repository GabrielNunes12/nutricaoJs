# Projeto de Nutrição usando Javascript puro

Tive essa ideia de criar um projeto que calculasse o IMC baseado no _peso(Kg)_ a _altura(m)_ e a _gordura corporal(%)_ e assim
se dá o IMC de uma pessoa!
